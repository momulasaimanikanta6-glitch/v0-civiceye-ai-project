# CivicEye AI India - User Manual

## Overview

CivicEye AI India is a multilingual AI-powered civic grievance assistance platform that helps citizens submit complaints through voice, text, image, and location inputs.

The system converts raw complaints into structured grievance tickets that are easier for government departments to process.

---

# Citizen Portal

## Step 1: Login

* Open the CivicEye AI India portal.
* Sign in using your registered account.

---

## Step 2: Create a Complaint

Click:

```text
Report Issue
```

You can submit complaints using:

### Text

Type your complaint in any supported language.

Example:

```text
Street lights are not working near the bus stop.
```

### Voice

* Click the microphone button.
* Speak naturally.
* The system converts speech to text.

### Image

Upload:

* Road damage photos
* Garbage accumulation
* Water leakage
* Public infrastructure issues

### Location

Allow location access or select the location manually.

---

## Step 3: AI Analysis

The system automatically:

* Converts voice to text
* Detects language
* Generates summary
* Identifies category
* Suggests department
* Assigns priority level

---

## Step 4: Review Ticket

Review:

* Complaint Summary
* Category
* Suggested Department
* Priority Score

Click:

```text
Submit
```

---

## Step 5: Track Complaint

Navigate to:

```text
My Complaints
```

You can view:

* Ticket ID
* Current Status
* Department Assigned
* Submission Date
* Resolution Updates

# Troubleshooting

### Voice not working

* Check microphone permissions.
* Refresh browser.

### Image upload failed

* Verify file size.
* Check internet connection.

### Location unavailable

* Enable GPS permissions.
* Enter location manually.
