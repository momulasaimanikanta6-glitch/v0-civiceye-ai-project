# Contributing to CivicEye AI India

Thank you for your interest in contributing to CivicEye AI India.

CivicEye AI India is an AI-powered multilingual civic grievance intelligence platform that helps citizens submit complaints through voice, text, image, and location while assisting administrators with intelligent routing, prioritization, and analytics.

## Team Members

| Role                 | Responsibility                                |
| -------------------- | --------------------------------------------- |
| Product Lead         | Documentation, project planning, presentation |
| Frontend Developer 1 | Citizen Portal                                |
| Frontend Developer 2 | Admin Dashboard                               |
| Backend Developer    | APIs, Database, Authentication                |
| AI/ML Engineer       | Speech Processing, NLP, Classification        |

---

## Development Workflow

### 1. Fork and Clone

```bash
git clone https://github.com/saimanikanta777/civiceye-ai.git
cd civiceye-ai-india
```

### 2. Create a Branch

```bash
git checkout -b feature/feature-name
```

Examples:

```bash
git checkout -b feature/voice-upload
git checkout -b feature/admin-dashboard
git checkout -b bugfix/login-api
```

### 3. Make Changes

Follow coding standards and project structure.

### 4. Commit Changes

```bash
git commit -m "Add complaint categorization API"
```

### 5. Push Changes

```bash
git push origin feature/feature-name
```

### 6. Open Pull Request

Create a Pull Request describing:

* Problem solved
* Changes made
* Screenshots (if UI changes)
* Testing performed

## Coding Standards

### Frontend

* React + Vite
* Tailwind CSS
* Functional Components
* Reusable Components
* Responsive Design

### Backend

* Flask/Django REST APIs
* RESTful Architecture
* Proper Error Handling
* JWT Authentication

### AI Layer

* Structured JSON Outputs
* Prompt Versioning
* Explainable Classification Results

## Code of Conduct

Be respectful and collaborative.

We welcome contributions that improve accessibility, multilingual support, grievance routing accuracy, analytics, and user experience.

Together, we can make civic grievance systems more accessible and intelligent for every citizen.
